# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security-sensitive reports.

Instead, email the maintainer with:

- A description of the issue
- Steps to reproduce
- Impact assessment (if known)

We will acknowledge receipt and aim to respond within a reasonable timeframe.

## Scope

This library performs local numerical computation on graph data supplied by the user. It does not run network services or handle credentials. Reports about optional dependencies (e.g. PyTorch, Jupyter) should reference the upstream package when appropriate.
